# Swiggy Marketplace Analytics

This project analyzes Swiggy restaurant and dish data to uncover insights about revenue trends, pricing strategies, and customer engagement.

## Objectives
- Identify top cities generating revenue
- Analyze pricing impact on ratings
- Explore customer engagement through rating counts
- Discover relationships between key variables using correlation analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Revenue strongly correlates with rating count, indicating customer engagement drives revenue.
- Bengaluru generates the highest estimated revenue among cities.
- Price shows minimal correlation with rating.

- Revenue by City
